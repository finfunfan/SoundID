How the audio works — the app never stores sound files. Recording metadata is pulled live from Xeno-canto's Creative-Commons collection via GBIF's open API (datasetKey b104…, no key, CORS-enabled), and the clip itself streams on demand into an <audio> element, so the device only holds the seconds currently playing. Only CC-licensed recordings are used, with recordist and licence shown on every clip. If a species has no fetchable clip (or the network blocks it), a short synthesised tone stands in so the loop never breaks. The dataProvider block still holds the eBird call for wiring real per-location frequencies later.
